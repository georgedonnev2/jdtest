# 安装指南

## 系统要求

- 操作系统: Windows 10+/macOS/Linux
- 内存: 至少 4GB RAM
- 存储空间: 至少 1GB 可用空间

## 安装步骤

1. **下载项目**
   ```bash
   git clone https://github.com/georgedonnev2/jdtest.git
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动应用**
   ```bash
   npm start
   ```
